# dev-python-desktop-kivy-hello-world

## Description
POC for kivy framework.

## Tech stack
- python
  - kivyMD

## Docker stack
- python:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
